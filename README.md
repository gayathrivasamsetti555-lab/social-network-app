# social-network-app
This project is a simple social network application.

Concepts Used:
- Sets: to store users
- Relations: follow connections between users
- Graph: represents the network
- BFS: used to recommend users

Features:
- Add user
- Follow users
- Find mutual friends
- Recommend connections

How it works:
Users are stored in a set.
Connections are stored in a graph.
BFS algorithm is used to suggest new connections.
